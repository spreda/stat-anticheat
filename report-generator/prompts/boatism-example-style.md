# BoatSim — Example Project Context (Reference Only)

This file documents the example project used in `content-md/report.md`.
Replace with your own project details.

## Project summary

- Unity 2018.4.14f1, C#, Windows
- Симулятор профессии судоводителя
- Реалистичная физика: плавучесть, вязкостное сопротивление, слэмминг
- Водомётный движитель, волновая поверхность (синусоиды + шум)
- Экономический контур: магазин, цены, баланс
- UI: скорость, время, день, магазин, 3 режима (E/F)
- Камера: вращение + масштабирование вокруг судна

## Special terms

- слэмминг (slamming) — ударные нагрузки при входе корпуса в воду
- водомётный движитель — реактивный движитель, направление струи меняет курс
- интерцептор — пластина на транце для дифферента
