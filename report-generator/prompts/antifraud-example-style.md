## Project summary

- Python 3.14, FastAPI, XGBoost, SQLite
- Система обнаружения и предотвращения мошенничества в игровых приложениях
- Векторизованное извлечение признаков из тиковых данных
- Классификатор XGBoost с оценкой риска 0--100
- Веб-интерфейс с загрузкой файлов и таблицей результатов
- Локальный запуск без подключения к сети

## Special terms

- мошенническое поведение -- использование игроком программного обеспечения, дающего нечестное преимущество
- градиентный бустинг -- ансамблевый метод машинного обучения
- тик -- минимальная единица времени в игре
- признак -- численная характеристика, извлекаемая из данных

## Technical stack

- FastAPI -- веб-фреймворк для асинхронной обработки запросов
- pandas -- библиотека для анализа данных
- XGBoost -- алгоритм машинного обучения на основе градиентного бустинга
- SQLite -- lightweight база данных для хранения результатов
- Jinja2 -- шаблонизатор для веб-интерфейса
